mobius.bico.options

Options

Help

[-h, --help]

Show the help message.

Dir

[]

Specify directory in which Bico does its job (there must be only one directory, and this argument is mandatory

Clazz

[]

generates also the file for this class, bico must be able to find it in its class path

Output

[-o]

The output directory.

Map

[-m, --map]

To have the map implementation.

List

[-l, --list]

To have the list implementation.

Lib

[-lib]

Enables the java.lang library generation. It implies the generation of most of the JRE.

ClassPath

[-cp]

The base working class path.
