The main package from bico. The class {@link mobius.bico.Main} is the class used to start everything. The packages herein are:
  • mobius.bico.dico handle correspondances between some Coq structures and the Java ones
  • mobius.bico.executors the classes which handle the translations.
  • mobius.bico.implem parts to handle bicolano lists and maps implementations
  • mobius.bico.visitors visitors to translate Java instructions to Bicolano instructions @see org.apache.bcel
