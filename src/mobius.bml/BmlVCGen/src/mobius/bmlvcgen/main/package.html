Classes in this package implement main functionality of BmlVCGen and provide entry points. To use bmlvcgen as a library, create and initialize and instance of {@link mobius.bmlvcgen.main.Env Env} class and use it to create a {@link mobius.bmlvcgen.main.BmlVCGen BmlVCGen} object.

Command-line tool functionality is implemented in {@link mobius.bmlvcgen.main.Main Main}.
