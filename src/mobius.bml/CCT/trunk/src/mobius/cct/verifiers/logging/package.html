This package contains classes related to logging. Logging is performed using objects implementing {@link mobius.cct.verifiers.logging.Logger Logger} interface. That interface contains methods used to log messages of various levels (errors, warnings, debug messages...). Default implementation ({@link mobius.cct.verifiers.logging.DefaultLogger DefaultLogger}) simply prints all messages to standard output stream, prefxing them with their levels. Most applications should probably create customized implementations of {@link mobius.cct.verifiers.logging.Logger Logger} integrated with their own logging mechanism.
