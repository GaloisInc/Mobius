Certificates

Class and method certificates are represented by objects of classes {@link mobius.cct.certificates.ClassCertificate ClassCertificate} and {@link mobius.cct.certificates.MethodCertificate MethodCertificate}. All certificates have the following properties:

  • Type (String)
  • Major and minor version numbers
  • Data (represented as a byte array)
Class certificates also contain a list of names of imported certificates.

Objects of class {@link mobius.cct.certificates.CertificatePack CertificatePack} contain a single class-level certificate and a collection of method certificates which hava a common type and version. {@link mobius.cct.certificates.CertificatePack CertificatePacks} may be created by using {@link mobius.cct.certificates.CertificatePackBuilder CertificatePackBuilders}.

Reading certificates

Certificates are read from the {@link mobius.cct.classfile.ClassFile ClassFile} by objects implementing the {@link mobius.cct.certificates.CertificateParser CertificateParser} interface. A default implementation is provided in class t{@link mobius.cct.certificates.DefaultCertificateParser DefaultCertificateParser}. Parsed certificates are exposed using a visitor API similar to that used to access class attributes in a {@link mobius.cct.classfile.ClassFile ClassFile}.

Writing certificates

Certified classes can be written using classes from package {@link mobius.cct.certificates.writer mobius.cct.certificates.writer}
