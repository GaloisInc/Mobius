Umbra Plugin

This plug-in provides the user the possibility to edit Java bytecode files equipped with BML specifications. It allows to transform Java source code into its bytecode textual representation similar to the one generated by javap utility. This can further be edited and saved as usual class file. Additionally this class file can be augmented with attributes which contain properties of the code expressed in the form of BML specifications.

This environment can be used as a part of MOBIUS Program Verification Environment and as a separate set of Eclipse plug-ins.
