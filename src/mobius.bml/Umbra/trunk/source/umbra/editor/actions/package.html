This package contains all the classes which handle the GUI actions available in the bytecode Umbra editor. This package contains two subpackages:

  • {@see umbra.editor.actions.history} which contains actions related to the handling of the historical versions of a byte code file.
  • {@see umbra.editor.actions.info} which contains actions related to the help information i.e. how to install the Umbra plug-in and how to use it.

The package itself contains the following classes to handle the byte code aditor actions:

  • {@see BytecodeColorAction} which handles the actions that change the colouring mode.
  • {@see BytecodeCombineAction} which handles the action that combines the changes.
  • {@see BytecodeRebuildAction} which handles the action that retrieves back the original, unedited version of the currently edited file.
  • {@see BytecodeRefreshAction} which handles the action that saves the current edits and reformats the current document in the default fashion.
  • {@see BytecodeSynchrAction} which handles the action to move from the byte code program point to corresponding point in the source code.

All these actions share certain functionality which is implemented in their common superclass {@see BytecodeEditorAction}.
