This package contains classes used to parse class files. It contains interfaces of class file implementations ({@link mobius.cct.repositories.classfile.ClassFile ClassFile} and {@link mobius.cct.repositories.classfile.MutableClassFile MutableClassFile}) as well as a simple implementation of these interfaces ({@link mobius.cct.repositories.classfile.DefaultClassFile DefaultClassFile}) and some utility classes. .

More details can be found in documentation of package {@link mobius.cct.repositories mobius.cct.repositories}.
