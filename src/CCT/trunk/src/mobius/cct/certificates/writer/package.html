Writing certificates

Class {@link mobius.cct.certificates.writer.CertificateWriter CertificateWriter} can be used to write certificates to a class. The certificates can be passed to it by calling appropriate visitor methods on it. A {@link mobius.cct.certificates.writer.CertificateWriter CertificateWriter} can be created from any instance of {@link mobius.cct.classfile.MutableClassFile MutableClassFile}. It removes all existing certificates from the class and writes new certificates to it.
