This package contains interfaces which must be implemented by objects representing class files.

Main interface is called {@link mobius.bmlvcgen.bml.ClassFile ClassFile}. It can provide a visitor with access to all information stored in a class file, including BML specifications attached to various class elements.

BML expressions can be accessed by providing an implementation of suitable kind of {@link mobius.bmlvcgen.bml.ExprVisitor ExprVisitor}.
