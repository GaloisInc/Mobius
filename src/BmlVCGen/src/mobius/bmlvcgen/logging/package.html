This package implements logging functions. All log entries are written through objects implementing the {@link mobius.bmlvcgen.logging.Logger Logger} interface. Such objects are created by {@link mobius.bmlvcgen.logging.LoggerFactory LoggerFactories}.

Two implementations of {@link mobius.bmlvcgen.logging.Logger Logger} are provided
  • {@link mobius.bmlvcgen.logging.SimpleLogger SimpleLogger}, which writes all messages to an output stream.
  • {@link mobius.bmlvcgen.logging.CompositeLogger CompositeLogger}, which passes all message to a collection of other loggers.

One implementation of {@link mobius.bmlvcgen.logging.LoggerFactory LoggerFactory} is provided, in class {@link mobius.bmlvcgen.logging.DefaultLoggerFactory DefaultLoggerFactory}. It creates loggers which write messages to stderr and to a file in current working directory.
