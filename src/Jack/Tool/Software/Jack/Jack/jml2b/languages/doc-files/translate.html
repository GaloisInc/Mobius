  • B_SUBSTRACTION_DOM
    This formula allows to formulate lemmas associated to assignable clause. It allows to formulate that a field is modified only on some instances.
    The left part corresponds to a set of modifiable instances.
    The right part can be:
    • B_FUNCTION_EQUALS
      The left and the right parts correspond to two functions that have to be identical excepted on some subset of their domain corresponding to the modifiable instances.
    • EQUALS_ON_OLD_INSTANCES
      The left part and the right parts correspond to two functions that have to identical only on the current instances set and excepted on some subset of their domain corresponding to the modifiable instances.
    • B_SUBSTRACTION_DOM
      In this case, they are mutliple set of modifiable instances, the defintion is recursive.
