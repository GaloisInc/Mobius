The Umbra plugin allows to edit the bytecode class files and view the BML specifications. The main package of the Umbra plugin contains only the general purpose classes that can be used everywhere in the sources or handle directly the lifecycle of the Umbra plugina as understood by Eclipse. The main package contains also all other Umbra packages as subpackages.

The subpackages of Umbra are:

  • {@see umbra.editor}
  • {@see umbra.instructions}
  • {@see umbra.java.actions}
  • {@see umbra.tests}
