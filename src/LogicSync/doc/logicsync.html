Description

This tool is part of the DirectVCGen component of the Mobius tool suite.

Usage

The program has to be called from its jar file with arguments having the following form:
Help | (Dir (Misc | Type | Clazz)*)

Options

Main options

Help
-h
-help
--help
Show the help message.
Dir
<file>
Specify directory in which Bico does its job (there must be only one directory, and this argument is mandatory).
Clazz
<string-list>
Generates also the file for the specified classes, bico must be able to find them in its class path.

Type options

Map
-m
-map
--map
Triggers the generation of the map implementation.
List
-l
-list
--list
Triggers the generation of the list implementation.

Misc options

Output
-o <file>
Used to specify the output directory.
ClassPath
-cp <string>
Specifies the base working class path.
Lib
-lib
Enables the java.lang library generation. It implies the generation of most of the JRE.
