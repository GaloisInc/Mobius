Configuring and Using the JML Plugin

Figure 3.4: Configuring the JML Checkers.
The JML plugin is configured on a per-project basis. The JML typechecker, compiler, and runtime assertion checker are configured independently. fig:jml_config shows the preferences panel for configuring the JML typechecker.

Figure 3.5: Executing the JML Compiler.
JML typechecking or compilation is currently accomplished via a menu action, as seen in fig:jmlc. In a future version of the Mobius PVE, autobuilders will be intelligently triggered instead.
