Mobius Program Verification Environment Help

"The Mobius PVE is a customized version of the Eclipse platform that integrates a set of tools that support software development using state-of-the-art formal methods and software engineering techniques. The Mobius PVE supports the specification and verification of JML-annotated Java programs at the source code level as well as at the bytecode level using logic- and type-based techniques and the Proof-Carrying Code paradigm."

It is composed of 4 parts:

  1. Program Verification Environment Guide

    It describes the overall architecture as well as the basic configurations steps.

  2. JML2 Plugin Guide

    A tool used for JML based runtime verification.

  3. ESC/Java2 Plugin Guide

    A tool used for JML based static verification.

  4. Umbra Guide

    A tool for Java bytecode annotations.
