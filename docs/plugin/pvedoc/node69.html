Executing the MOBIUS Prover Editor

The MOBIUS Prover Editor is made out of two panels Fig. 3.12. It is automatically loaded and initialized anytime a Coq theory file or proof script file is loaded within the Mobius PVE. No special configuration is necessary. If the prover needs to be reset, a small reinitialize button is available in the toolbar.
